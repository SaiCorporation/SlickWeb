# Code Directory 
<p>Open Development Base = `load dtb y/n odb`</p>
<p>Load console.gxe = `open c.gxe n/a`</p>
<p>=============================================Server Commands=============================================</p>
<p>Reboot Server = `Serv rboot wait.3s o/e`</p>
<p>Reload Randomized Server Search = `Rload RSS o/e`</p>
<p>Power on Server `Serv pwr.on load vroot r/b`</p>
<p>Power off Server `Serv pwr.off load vroot r/b`</p>
